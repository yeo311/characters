# Character List Web App

## How to run

```
npm run deploy
```

or

```
yarn deploy
```

## Framework & Library

- React.js
  - 해당 프로젝트의 경우, 싱글 페이지로 구성되어 있고, 캐릭터 정보 컴포넌트의 반복적인 재사용을 위하여 React.js 사용
- Redux
  - 캐릭터 리스트, 페이지, 필터 등의 상태 관리를 위해 Redux 사용
- Redux-Thunk
  - API를 비동기적으로 호출하기 위해 Redux 미들웨어인 Redux-Thunk 사용
